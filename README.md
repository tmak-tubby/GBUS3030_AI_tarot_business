# ✦ ARCANA AI: AI Spiritual Home

Arcana AI is a frontend prototype for an AI-powered spiritual wellness platform. Positioned as a "Spiritual Home," it moves beyond simple chatbot tarot readings by deeply integrating with a user's emotional data to provide highly personalized, context-aware guidance.

## The Problem
Facing academic pressure, a competitive job market, loneliness, and relationship anxieties, young adults are turning to mystical practices (Tarot, Bazi, Face Reading) for emotional comfort. However, current digital solutions are often sterile, lacking the emotional depth and continuity of an in-person expert.

## The Solution & Business Model
Lumina utilizes a proprietary data loop to act as a longitudinal emotional guide. 

**Core Features:**
* **The Fortune Diary:** Users log their daily anxieties, hopes, and events (via text or voice note). This creates a localized context model. Integrates with Apple Journal, Notion, and Google Calendar.
* **The Reading Room:** An interactive digital deck (or a companion for physical decks) where users consult the cards. The AI cross-references the card pulls with the user's Fortune Diary entries to provide incredibly specific, emotionally resonant readings.

**Monetization (Freemium):**
* **Seeker ($0/mo):** 1 daily tarot pull, basic diary logging, standard AI chat.
* **Mystic ($8.99/mo):** Unlimited readings, full diary pattern analysis, unlocks advanced services (Bazi, Face & Palm Reading), and full API syncing with external journals.

## About This Prototype
This repository is a static frontend mockup built with HTML, CSS, and Vanilla JavaScript. 
* Features an interactive CSS 3D-flip 78-card deck.
* Utilizes a "Wizard of Oz" JS script to simulate the AI's contextual awareness (linking diary entries to specific tarot suits) for user testing without requiring an active LLM backend.
